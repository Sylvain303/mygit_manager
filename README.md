# `mygit_manager`

manage your personal git repos

code inspired from
https://github.com/agateau/reposetup

## LICENCE

GPLv3

## Install

~~~
git clone mygit_manager
cd mygit_manager
make install.sh
~~~

## Usage

Add a repos (same behavior)
~~~
mygit_manager add REPOS_NAME
mygit_manager create REPOS_NAME
~~~

List repos

~~~
mygit_manager list
~~~

remove a repos

~~~
mygit_manager rm REPO_NAME
~~~
