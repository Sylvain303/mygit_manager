# `mygit_manager`

manage your personal git repos

## Install

~~~
git clone mygit_manager
cd mygit_manager
make install.sh
~~~

## Usage

Add a repos
~~~
mygit_manager add repos_name
~~~

List repos

~~~
mygit_manager list
~~~
